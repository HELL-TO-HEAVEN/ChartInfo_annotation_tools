This is the code for the annotation tools used to create the PubMedCentral chart dataset used for ICDAR 2019 CHART-Infographics. The code is based on existing API's previously released by Kenny Davila and Richard Zanibbi at the DPRL in RIT, USA. The provided tools were developed at the University at Buffalo in Buffalo, New York, USA, in the Center for Unified Biometrics and Sensors (CUBS).
